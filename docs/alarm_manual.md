# DisplayEdu Fab 교육용 가상 알람 매뉴얼

> 본 문서는 AI Agent 교육을 위한 가상 알람 매뉴얼입니다. 실제 기업의 설비, 공정, 알람 기준, 조치 절차와 무관합니다.

---

## 1. 문서 목적

이 문서는 삼성디스플레이 재직자 대상 AI Agent Architecture 강의에서 사용할 **교육용 가상 알람 매뉴얼**입니다.

수강생은 이 문서를 통해 AI Agent가 알람 로그만 보고 임의로 답변하지 않고,  
문서 근거를 참고하여 다음 내용을 정리하는 흐름을 실습할 수 있습니다.

- 알람 코드의 의미 확인
- 반복 발생 여부 판단
- 관련 설비와 챔버 확인
- 원인 후보 정리
- 1차 확인 항목 도출
- 권장 조치 방향 정리
- 품질 영향 가능성 설명
- 추가 확인 필요 사항 제시

이 문서의 모든 설비, 라인, 알람 코드, 수치, 판단 기준은 교육용 가상 예시입니다.

---

## 2. 교육용 가상 시나리오 개요

| 항목 | 내용 |
|---|---|
| 가상 회사명 | DisplayEdu Fab |
| 가상 라인명 | EDU-LINE-07 |
| 가상 공정 | Thin Film Deposition |
| 가상 설비 ID | EQP-EV-03 |
| 주요 가상 알람 코드 | ALM-TEMP-402 |
| 주요 상황 | 박막 증착 공정 중 챔버 온도 편차 관련 알람이 반복 발생함 |
| Agent 목표 | 알람 로그와 매뉴얼 근거를 바탕으로 반복 발생 여부, 심각도, 관련 설비, 현장 메모, 원인 후보, 조치 방향을 정리함 |

이 시나리오는 실제 제조 현장을 그대로 재현하기 위한 것이 아닙니다.  
AI Agent가 제조 로그와 기술 문서를 함께 참고하는 구조를 이해하기 위한 교육용 예시입니다.

## 3. ALM-TEMP-402 알람 개요

### 3.1 알람 코드

- 알람 코드: `ALM-TEMP-402`
- 알람 이름: 교육용 챔버 온도 편차 알람
- 관련 공정: Thin Film Deposition
- 관련 설비: EQP-EV-03
- 주요 키워드: 온도 편차, 반복 발생, 챔버, 박막 균일도, 품질 영향 가능성, 추가 확인 필요

### 3.2 알람 의미

`ALM-TEMP-402`는 교육용 가상 시나리오에서  
박막 증착 공정 중 특정 챔버의 온도 값이 교육용 기준 범위에서 벗어난 상황을 나타내는 알람입니다.

이 알람은 단독으로 한 번 발생했을 때보다,  
같은 설비와 같은 챔버에서 여러 번 반복 발생할 때 더 주의 깊게 확인할 수 있습니다.

AI Agent는 이 알람을 해석할 때 다음 정보를 함께 확인할 수 있습니다.

- 같은 알람 코드가 몇 번 발생했는지
- 같은 설비에서 반복 발생했는지
- 같은 챔버에서 집중적으로 발생했는지
- 심각도 값이 `WARNING`인지 `CRITICAL`인지
- 현장 작업자 메모에 반복, 재발, 확인 필요 등의 표현이 있는지

## 4. ALM-TEMP-402 발생 가능 상황

`ALM-TEMP-402`는 교육용 시나리오에서 다음과 같은 상황을 설명하기 위해 사용됩니다.

### 4.1 챔버 온도 편차 발생

Thin Film Deposition 공정 중 챔버 온도가 교육용 기준 범위에서 벗어난 것으로 기록될 수 있습니다.

이 경우 Agent는 단순히 “온도 문제가 발생했다”고 답하기보다,  
로그에서 측정값, 발생 시각, 반복 횟수, 심각도, 현장 메모를 함께 확인해야 합니다.

### 4.2 같은 챔버에서 반복 발생

같은 챔버에서 `ALM-TEMP-402`가 여러 번 발생했다면  
단순 일회성 알람이 아니라 반복 발생 패턴으로 볼 수 있습니다.

이 경우 다음과 같은 표현을 사용할 수 있습니다.

- “동일 챔버에서 반복 발생한 것으로 확인할 수 있습니다.”
- “반복 발생 여부에 대한 추가 확인이 필요합니다.”
- “일시적 알람인지, 특정 챔버 중심의 재발 패턴인지 구분할 수 있습니다.”

### 4.3 다른 알람과 함께 발생

온도 편차 알람이 진공 상태 변동, 압력 변동, 가스 흐름 확인 알람과 함께 나타날 수 있습니다.

관련 교육용 알람 예시는 다음과 같습니다.

- `ALM-VAC-210`: 교육용 진공 상태 변동 알람
- `ALM-PRESS-115`: 교육용 챔버 압력 변동 알람
- `ALM-FLOW-301`: 교육용 가스 흐름 확인 알람

이러한 알람이 함께 나타난 경우, Agent는 원인을 단정하지 말고  
“연관 가능성이 있으므로 추가 확인이 필요합니다”라고 표현해야 합니다.

## 5. ALM-TEMP-402 1차 확인 항목

`ALM-TEMP-402`가 발생했을 때 Agent는 다음 항목을 우선 확인할 수 있습니다.

### 5.1 로그 기준 확인 항목

| 확인 항목 | 설명 |
|---|---|
| 발생 시각 | 특정 시간대에 집중적으로 발생했는지 확인할 수 있습니다. |
| 설비 ID | `EQP-EV-03`에서 반복 발생했는지 확인할 수 있습니다. |
| 챔버 ID | 특정 챔버에서 집중적으로 발생했는지 확인할 수 있습니다. |
| 알람 코드 | `ALM-TEMP-402`가 여러 번 반복되었는지 확인할 수 있습니다. |
| 심각도 | `WARNING`, `CRITICAL` 수준을 구분할 수 있습니다. |
| 반복 횟수 | `repeat_count` 값을 참고해 반복 발생 여부를 판단할 수 있습니다. |
| 현장 메모 | 작업자가 남긴 메모에서 재발, 확인 필요, 관찰 등의 표현을 확인할 수 있습니다. |

### 5.2 문서 기준 확인 항목

Agent는 알람 매뉴얼에서 다음 키워드를 검색할 수 있습니다.

- 온도 편차
- 반복 발생
- 챔버
- 박막 균일도
- 품질 영향 가능성
- 추가 확인 필요
- 원인 후보
- 권장 조치

이 키워드는 1일차 단순 문서 검색 실습과 2일차 RAG 검색 실습에서 사용할 수 있습니다.

## 6. ALM-TEMP-402 원인 후보

`ALM-TEMP-402`의 원인은 이 문서만으로 단정할 수 없습니다.  
다만 교육용 시나리오에서는 다음 항목을 원인 후보로 정리할 수 있습니다.

### 6.1 챔버 온도 제어 상태 확인 필요

챔버 온도 값이 반복적으로 교육용 기준 범위 밖으로 기록되었다면,  
온도 제어 상태를 점검 대상으로 볼 수 있습니다.

Agent 답변 예시는 다음과 같습니다.

> “로그상 동일 챔버에서 온도 편차 알람이 반복 발생한 것으로 확인됩니다. 온도 제어 상태는 점검 대상으로 볼 수 있으며, 추가 확인이 필요합니다.”

### 6.2 센서 값 변동 가능성 확인 필요

측정값이 일정하지 않고 반복적으로 흔들리는 형태라면,  
센서 값 변동 가능성을 확인할 수 있습니다.

단, Agent는 센서 이상이라고 단정하면 안 됩니다.

권장 표현은 다음과 같습니다.

- “센서 값 변동 가능성을 확인할 수 있습니다.”
- “센서 이상으로 단정하기보다는 측정값 추이를 추가 확인할 수 있습니다.”
- “반복 로그와 현장 메모를 함께 검토할 수 있습니다.”

### 6.3 챔버 주변 상태 변화 가능성 확인 필요

온도 편차가 특정 챔버에 집중되면,  
해당 챔버 주변 상태 변화가 영향을 주었는지 확인할 수 있습니다.

이 경우에도 실제 공정 조건이나 실제 설비 세팅값은 언급하지 않습니다.

### 6.4 다른 교육용 알람과의 연관 가능성

`ALM-VAC-210`, `ALM-PRESS-115`, `ALM-FLOW-301` 같은 다른 교육용 알람이 근접한 시간대에 발생했다면  
온도 편차 알람과 함께 참고할 수 있습니다.

다만 Agent는 다음과 같이 조심스럽게 표현해야 합니다.

> “동일 시간대에 다른 교육용 알람도 일부 확인되므로, 온도 편차 알람과의 연관 가능성은 추가 확인이 필요합니다.”

## 7. ALM-TEMP-402 권장 조치

이 문서의 권장 조치는 실제 작업 지시가 아니라 교육용 판단 흐름입니다.  
Agent는 실제 조치를 명령하지 않고, 확인 방향을 제안하는 형태로 답변해야 합니다.

### 7.1 반복 발생 여부 확인

먼저 같은 설비, 같은 알람 코드, 같은 챔버에서 알람이 반복 발생했는지 확인할 수 있습니다.

확인 예시는 다음과 같습니다.

- `equipment_id`가 `EQP-EV-03`인지 확인
- `alarm_code`가 `ALM-TEMP-402`인지 확인
- `chamber_id`가 동일한지 확인
- `repeat_count`가 증가하는지 확인
- `severity`에 `CRITICAL`이 포함되는지 확인

### 7.2 현장 메모 확인

`operator_note`에 다음과 같은 표현이 있는지 확인할 수 있습니다.

- 반복
- 재발
- 확인 필요
- 검토 필요
- 추가 관찰
- 동일 챔버

이러한 표현이 여러 번 나타나면, Agent는 반복 발생 가능성을 더 명확하게 설명할 수 있습니다.

### 7.3 관련 교육용 알람 함께 확인

온도 편차 알람만 보지 않고, 주변 시간대의 다른 교육용 알람도 함께 확인할 수 있습니다.

- `ALM-VAC-210`: 진공 상태 변동 관련 참고 알람
- `ALM-PRESS-115`: 챔버 압력 변동 관련 참고 알람
- `ALM-FLOW-301`: 가스 흐름 확인 관련 참고 알람

단, 다른 알람이 있다고 해서 원인을 단정하지 않습니다.  
Agent는 “함께 확인할 수 있습니다”, “추가 확인이 필요합니다”와 같이 표현합니다.

### 7.4 리포트 작성

Agent는 다음 구조로 대응 리포트를 작성할 수 있습니다.

1. 알람 요약
2. 반복 발생 여부
3. 심각도 요약
4. 관련 설비와 챔버
5. 로그 근거
6. 매뉴얼 근거
7. 원인 후보
8. 권장 확인 항목
9. 추가 확인 필요 사항

## 8. ALM-TEMP-402 품질 영향 가능성

`ALM-TEMP-402`는 교육용 시나리오에서 챔버 온도 편차와 관련된 알람입니다.  
박막 증착 공정에서 온도 편차가 반복적으로 발생하면, 교육용 관점에서는 박막 균일도와 품질 영향 가능성을 함께 검토할 수 있습니다.

다만 이 문서는 실제 품질 기준이나 실제 수율 기준을 포함하지 않습니다.  
따라서 Agent는 다음과 같이 표현해야 합니다.

- “박막 균일도에 영향을 줄 가능성이 있는지 추가 확인이 필요합니다.”
- “품질 영향 가능성을 검토 대상으로 볼 수 있습니다.”
- “실제 품질 영향 여부는 별도의 품질 데이터 확인이 필요합니다.”
- “이 알람만으로 품질 이상을 단정할 수 없습니다.”

Agent는 알람 로그만 보고 실제 품질 문제를 확정해서는 안 됩니다.  
품질 영향 가능성은 어디까지나 추가 확인이 필요한 항목으로 표현해야 합니다.

## 9. Agent가 답변할 때 주의할 점

AI Agent가 `ALM-TEMP-402`에 대해 답변할 때는 다음 원칙을 따라야 합니다.

### 9.1 단정하지 않기

알람 로그와 매뉴얼만으로 실제 원인을 확정하면 안 됩니다.

부적절한 표현 예시는 다음과 같습니다.

- “원인은 온도 제어 장치 고장입니다.”
- “품질 불량이 발생했습니다.”
- “즉시 특정 조치를 수행해야 합니다.”

적절한 표현 예시는 다음과 같습니다.

- “온도 제어 상태를 점검 대상으로 볼 수 있습니다.”
- “품질 영향 가능성은 추가 확인이 필요합니다.”
- “반복 발생 여부를 먼저 확인할 수 있습니다.”

### 9.2 근거를 함께 제시하기

Agent는 답변할 때 로그 근거와 매뉴얼 근거를 함께 제시해야 합니다.

예시는 다음과 같습니다.

> “로그에서 `EQP-EV-03` 설비의 `ALM-TEMP-402`가 여러 차례 발생했고, 매뉴얼에서는 이 알람을 챔버 온도 편차와 관련된 교육용 알람으로 설명하고 있습니다.”

### 9.3 교육용 데이터임을 유지하기

Agent 답변에는 실제 기업 내부 기준, 실제 설비 세팅값, 실제 공정 조건, 실제 조치 절차처럼 보이는 내용을 포함하지 않아야 합니다.

적절한 표현은 다음과 같습니다.

- “교육용 기준 범위”
- “가상 설비”
- “추가 확인 필요”
- “점검 대상으로 볼 수 있음”
- “가능성을 검토할 수 있음”

### 9.4 민감하거나 실제적인 내부 정보처럼 보이는 표현 피하기

다음 표현은 사용하지 않습니다.

- 실제 내부 시스템명
- 실제 설비 제조사명
- 실제 공정 레시피
- 실제 허용오차
- 실제 품질 기준
- 실제 조치 절차명
- 실제 생산량 또는 수율 수치

## 11. Agent 답변 예시

아래 예시는 교육용 문서 검색과 RAG 실습에서 사용할 수 있는 답변 형식입니다.

### 11.1 짧은 요약 예시

`EQP-EV-03` 설비에서 `ALM-TEMP-402`가 반복 발생한 것으로 확인할 수 있습니다.  
이 알람은 교육용 매뉴얼에서 챔버 온도 편차와 관련된 가상 알람으로 설명됩니다.  
동일 챔버에서 반복 발생했는지, 심각도에 `CRITICAL`이 포함되는지, 현장 메모에 재발 또는 확인 필요 표현이 있는지 추가 확인할 수 있습니다.

### 11.2 리포트형 답변 예시

#### 알람 요약

- 설비 ID: EQP-EV-03
- 알람 코드: ALM-TEMP-402
- 알람 의미: 교육용 챔버 온도 편차 알람
- 관련 공정: Thin Film Deposition
- 주요 키워드: 온도 편차, 반복 발생, 챔버, 품질 영향 가능성

#### 반복 발생 여부

로그에서 동일 설비와 동일 알람 코드가 여러 번 확인되면 반복 발생으로 볼 수 있습니다.  
특히 같은 챔버에서 반복적으로 기록되는 경우, 일회성 알람보다 추가 확인이 필요한 상황으로 정리할 수 있습니다.

#### 원인 후보

- 챔버 온도 제어 상태 확인 필요
- 센서 값 변동 가능성 확인 필요
- 챔버 주변 상태 변화 가능성 확인 필요
- 다른 교육용 알람과의 연관 가능성 확인 필요

#### 권장 확인 항목

- 동일 설비 반복 여부
- 동일 챔버 집중 여부
- 심각도 변화
- 현장 메모
- 주변 시간대의 관련 교육용 알람
- 품질 영향 가능성

#### 주의 사항

이 답변은 교육용 가상 데이터와 가상 매뉴얼을 바탕으로 한 예시입니다.  
실제 설비 조치나 실제 공정 판단으로 사용하면 안 됩니다.
